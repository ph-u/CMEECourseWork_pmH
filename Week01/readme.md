### CompileLaTeX.sh

#### Features

make LaTeX script (with `bib` bibliography file) into pdf; `TeX` and `bib` files must have same filename

#### Input

```
./CompileLaTeX.sh <tex without extension>
```

#### Output

saves the output into a .pdf file in `Data` subdirectory
*****

#### Features

#### Input
